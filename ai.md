## Assistant vs Agent

| 概念                | 特点                            | 举例                                   |
| ----------------- | ----------------------------- | ------------------------------------ |
| **Assistant（助手）** | 比较泛、通常指一个面向用户的智能体，用于对话、帮你做事。  | ChatGPT、Siri、Copilot                 |
| **Agent（智能体）**    | 更技术化，能**自主规划、执行动作、调用工具**完成目标。 | AutoGPT、Claude Agent、LangChain Agent |

- Claude Code里是智能体 ,Cherry Studio 里是助手
- 所有的 **agent 都可以是 assistant**， 但不是所有的 **assistant 都是 agent**。
